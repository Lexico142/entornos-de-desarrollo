# Sistema de Login - Campus Digitech

Este programa simula la entrada a una página web con usuario y contraseña.

## Funcionamiento básico

1.  **Bucle:** El programa nunca se acaba por sí solo (usa un "while true"), siempre vuelve a pedirte el login a menos que tú elijas "Salir".

2.  **El Login:**
    * Te pide nombre y clave.
    * Revisa una lista guardada en el código.
    * Si aciertas, te deja pasar. Si fallas, te lo vuelve a pedir.

3.  **El Menú:**
    * Si entras, puedes ver opciones (incompletas).
    * **Cerrar sesión (Opción 3):** Te borra los datos y te devuelve a la pantalla de login.
    * **Salir (Opción 4):** Aquí sí se cierra el programa del todo.
